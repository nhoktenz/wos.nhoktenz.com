# Canyon Clash Team Divider 🏜️

A simple tool to split players into 3 balanced Canyon Clash teams based on power.

## Live Page

- https://wos.nhoktenz.com/canyon_clash

## Features

- Add players with name and power
- Remove players from the list
- Live totals:
  - Total members
  - Total power
- Auto-balance into 3 teams
- Team summary per team:
  - Member count
  - Total power
  - Average power

## How It Works

1. Add player name and power.
2. Repeat until all players are added.
3. Click "Divide into Teams".
4. The app sorts players by power and assigns each player to the current lowest-power team.

This greedy balancing keeps team power as close as possible in most practical cases.
